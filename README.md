# 🏢 NGO ERP - Complete Management System

A comprehensive web-based ERP system for NGOs with multi-user role-based access, offline support, and PWA capabilities.

## 📋 Features

### Core Modules
- **Beneficiary Management** – Track beneficiaries with QR codes
- **Donation Management** – Record donations, generate receipts
- **Expense Management** – Track expenses by category
- **Program Management** – Manage NGO programs and budgets
- **Volunteer Management** – Coordinate volunteers and hours
- **Document Management** – Store and organize documents

### Advanced Features
- **Multi-User Roles** – Admin, Finance, Program Manager, Volunteer, Guest, etc.
- **Offline Support** – Works without internet, syncs when online
- **PWA Ready** – Install as native app on mobile/desktop
- **Analytics Dashboard** – Charts, trends, AI predictions
- **Calendar** – Event scheduling and reminders
- **Communications** – Messages, announcements, newsletters
- **Export Reports** – CSV/Excel export for all modules

## 🚀 How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/A-GESFETHA/NGO-ERP-System.git
   
2.  **Open** index.html in your browser or use Live Server in VS Code

3. **For Firebase Integration:**
   1. Create a Firebase project
   2. Enable Authentication and Firestore
   3. Add your Firebase config to a firebase-config.js file
